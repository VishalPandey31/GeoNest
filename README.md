# GeoNest 🏡

GeoNest is a premium, dark-themed real estate platform for the Mumbai, Navi Mumbai, and Thane regions. It features dynamic property listings, robust location-based search, a lead-generation system via WhatsApp, and an AI-powered property assistant.

## 🚀 Live Preview

**[View the Live Project Here](https://GeoNest.surge.sh)**

## Features

- **Modern UI**: Dark-themed, MagicBricks-style interface.
- **Smart Search**: Find properties across 60+ locations.
- **AI Assistant**: Powered by Gemini for smart real estate and property recommendations.
- **Lead Generation**: Seamless WhatsApp integration for property inquiries.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Navigation**: React Router DOM 
- **AI Integration**: Google Generative AI (`@google/generative-ai`)
- **Icons**: Lucide React
